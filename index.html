bun is like postcss, babel, node & webpack in one 100x faster tool for building modern web frontends.

Request early access

for macOS (x64 & Apple Silicon) and Linux x64




more precisely, bun is a very new:

  • jsx & typescript transpiler
  • js & css bundler
  • dev server with 60fps hot module reloading
  • javascript runtime environment (like node.js or deno)

you can use bun with Next.js and Create React App. bun does not support AST plugins yet, but it will.

bun is written in Zig, a low-level LLVM frontend (like Rust) that compiles to native code.
